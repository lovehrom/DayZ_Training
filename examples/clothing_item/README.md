# Clothing Item Example

## Что это
Пример создания Clothing предмета (одежда) со слотами для attachments.

## Файлы
- `config.cpp` - Конфигурация со слотами (Armband, Shoulder)
- `scripts/3_Game/Items/Shirt_Custom.c` - Класс одежды

## Ключевые моменты
- **Inventory slots** - инвентарь одежды
- **Attachments** - слоты для accessories (нашивки, патронташ)
- **OnAttachment()** - логика при присоединении
