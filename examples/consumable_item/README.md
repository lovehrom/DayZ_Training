# Consumable Item Example

## Что это
Пример создания Consumable предмета (расходник) для DayZ.

## Как использовать
1. Скопируйте мод в `@ConsumableItem/`
2. Добавьте `-mod=@ConsumableItem`
3. Запустите сервер

## Файлы
- `config.cpp` - Конфигурация Battery
- `scripts/3_Game/Items/Battery9V.c` - Класс батарейки
