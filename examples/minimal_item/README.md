# MinimalItem Example

## Что это

Минимальный пример предмета для DayZ. Самый простой рабочий мод.

## Как использовать

1. **Скопировать мод:**
   - Создайте папку `@MinimalItem/` рядом с `@DayZServer/`
   - Скопируйте файлы мода в `@MinimalItem/`

2. **Запустить сервер:**
   - Добавьте параметр запуска: `-mod=@MinimalItem`
   - Запустите сервер

3. **Проверить в игре:**
   - Зайдите на сервер
   - Используйте `_test = "MinimalItem"_spawn` для спавна
   - Предмет должен появиться

## Файлы

- `config.cpp` - Конфигурация мода (определение предмета)
- `scripts/3_Game/Items/MinimalItem.c` - Код предмета

## Структура

```
@MinimalItem/
├── config.cpp
└── scripts/
    └── 3_Game/
        └── Items/
            └── MinimalItem.c
```

## Известные ограничения

- Использует заглушку модели (apple)
- Не имеет уникальной иконки
- Не имеет кастомных звуков
