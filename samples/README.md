# Official DayZ Samples

Эта папка содержит официальные примеры от Bohemia Interactive.

## Как получить samples

**Вариант 1: Из Steam Workshop**
1. Откройте DayZ Tools
2. Перейдите in "Workshop Files"
3. Найдите "DayZ Samples"
4. Скачайте

**Вариант 2: Из GitHub**
```bash
git clone https://github.com/BohemiaInteractive/DayZ-Samples.git samples/official
```

## Почему samples не в репозитории?

- **Размер:** ~1.1 GB
- **Дублирование:** Уже доступны в Steam Workshop и GitHub
- **Экономия:** git clone в 10+ раз быстрее

## Что внутри


## Как использовать

Скопируйте нужный пример в свой проект и адаптируйте под свои нужды.
